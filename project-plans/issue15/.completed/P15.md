### Reducer Fix (Missing State Transition)

#### `src/state/mod.rs` — `SelectRepository` handler

Added logic: when `SelectRepository(idx)` fires while `issues_state.active == true`, the reducer now:
- Discards any active inline draft with a `draft_notice` set to "Unsent draft discarded"
- Clears `issues_state.issues`, `selected_issue_index`, `issue_detail`, `list_cursor`
- Resets `has_more_issues = false` and `list_loading = true`
- Clears `error`

This implements the scope-change invalidation required by REQ-ISS-001 and REQ-ISS-013.
